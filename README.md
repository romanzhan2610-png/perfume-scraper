# Парсер парфюмерного каталога (Undetected Selenium)

Этот репозиторий содержит продвинутый, отказоустойчивый веб-парсер, созданный для извлечения массивного датасета (~120 000 товаров, 7 500 брендов) из глобального парфюмерного каталога. 

Целевой ресурс использует агрессивные механизмы защиты от ботов, включая проверку целостности браузера от Cloudflare и жесткое лимитирование запросов (HTTP 429 Too Many Requests).

## Архитектура и инженерные решения

При подготовке к миграции ~120k записей в среду WooCommerce использование существующих open-source датасетов было отклонено. Основные причины: неактуальность данных, плохая нормализация и отсутствие прямых URL-адресов товаров, необходимых для дальнейшего обогащения базы (парсинг нот и пирамид). 

Был разработан кастомный парсер со следующим функционалом:

* **Обход антифрод-систем:** Использование `SeleniumBase` в режиме `uc=True` (Undetected Chromedriver) для обхода проверок Cloudflare turnstiles и маскировки отпечатков webdriver.
* **Очистка данных «на лету» (Data Cleaning):** Извлеченные сырые строки динамически парсятся с помощью регулярных выражений (RegEx) для разделения названия продукта, года выпуска и гендерной принадлежности. Имя бренда автоматически вырезается из названия самого аромата для обеспечения чистоты БД при импорте.
* **Отказоустойчивое хранение:** Данные сохраняются в формате JSON Lines (`.jsonl`) в режиме добавления (append-only). Это гарантирует сохранность данных при неожиданных сбоях ОС или блокировках со стороны сервера.
* **Два режима работы:** 
   * **Manual Turbo Mode:** Создан для максимальной скорости под контролем оператора. Использует `page_load_strategy="eager"` для перехвата DOM до подгрузки тяжелых медиа-ассетов. При получении ошибки 429 скрипт уходит в паузу, ожидая ручного переключения ноды VPN на уровне ОС, после чего бесшовно продолжает работу.
    * **Auto Stealth Mode:** Предназначен для автономной работы (например, ночью). Включает динамическую генерацию Chrome-расширений для аутентификации приватных прокси. Имитирует поведение живого пользователя (скроллинг, движение мыши, случайные задержки) и автоматически ротирует прокси-ноды при обнаружении бана.

## Стек технологий
* Python 3.9+
* SeleniumBase (Undetected Mode)
* BeautifulSoup4 (Парсинг DOM)
* RegEx (Нормализация данных)
* Telegram API (Система алертов и мониторинга)

## Запуск

1. Клонируйте репозиторий и установите зависимости:
   `pip install -r requirements.txt`
2. Для высокоскоростного парсинга под контролем (требует ручного переключения VPN):
   `python scraper_manual_turbo.py`
3. Для автономного парсинга в фоне (требует настройки реквизитов прокси в конфигурации скрипта):
   `python scraper_auto_stealth.py`
